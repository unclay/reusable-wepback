# reusable-wepback

可复用的webpack配置文件

# 功能支持

+ webpack
+ vue
+ less
+ img 2 base64
+ es2015

# 目录

+ examples
  + webpack-base
+ build
  + webpack.base.conf.js 基础的配置
  + webpack.prod.conf.js 用于打包文件的配置
  + dev-server.js        开发服务器配置
  + hot-server.js        热加载配置
  + html-plugin.js       html文件配置
  + uglify-js-plugin.js  打包压缩js配置
